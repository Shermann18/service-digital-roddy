# Digital Roddy - Services Digitaux Madagascar

Site web professionnel pour présenter mes services digitaux à Madagascar.

## Services offerts :
- Présentations PowerPoint professionnelles
- Automatisation Excel & Dashboards
- Création de logos & identité visuelle
- Projets Arduino & IoT
- Formation Office 365
- Documents Word professionnels

## Technologies utilisées :
- HTML5
- CSS3 (Flexbox, Grid, Variables CSS)
- JavaScript Vanilla
- Responsive Design
- Font Awesome Icons
- Google Fonts

## Installation :
1. Téléchargez les fichiers
2. Ouvrez `index.html` dans votre navigateur
3. Modifiez `script.js` avec vos coordonnées

## Configuration :
Modifiez le fichier `script.js` avec vos informations :
- Votre numéro WhatsApp
- Votre email
- Vos images dans le dossier `images/`

## Fonctionnalités :
- Menu responsive
- Formulaire de commande WhatsApp
- Portfolio avec exemples
- Modal interactive
- Animations CSS
- Smooth scrolling
